
# Aura AI Companion (Hinglish AI GF)

This project is a virtual companion app styled like an Instagram Chat, powered by **Gemini 3 Flash**.

## How to open in Visual Studio Code

1.  **Create a folder**: Create a new folder on your computer named `aura-ai-gf`.
2.  **Copy Files**: Copy all the files provided in this code response into that folder.
3.  **Open VS Code**: 
    *   Open Visual Studio Code.
    *   Go to `File` > `Open Folder...` and select your `aura-ai-gf` folder.
4.  **Install Dependencies**:
    *   Open the terminal in VS Code (`Ctrl + ~` or `Cmd + ~`).
    *   Type `npm install` and press Enter.
5.  **Run the App**:
    *   Type `npm run dev` in the terminal.
    *   Click the link provided (usually `http://localhost:5173`) to see your AI GF in the browser.

## Features
- **Hinglish Support**: Chat in a mix of Hindi and English.
- **Instagram UI**: Real-time typing indicators and chat bubbles.
- **Gemini 3 Flash**: High-speed, empathetic responses.
